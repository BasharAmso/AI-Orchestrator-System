# My Project

> Built with [Bashi](https://github.com/BasharAmso/bashi)

## Getting Started

Open this project in VS Code with Claude Code, then:

```
/start             See where you are and what to do next
/setup             Set your project name, stack, and purpose
/capture-idea      Describe what you want to build
/run-project       Start building
```

## Commands

| Command | What It Does |
|---------|-------------|
| `/start` | Shows current status and suggests next action |
| `/run-project` | Executes the next task |
| `/save` | Saves progress for next session |
| `/status` | Quick snapshot of progress |
| `/doctor` | Diagnose issues |
